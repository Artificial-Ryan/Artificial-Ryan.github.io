A Python-based tool that captures and analyzes network packets in real-time. Features a user-friendly graphical interface built with Tkinter and leverages the power of Scapy for packet manipulation.

A secure solution for managing passwords, built with Python. This tool uses robust cryptographic techniques to ensure that your sensitive information is stored safely and can be accessed easily through its intuitive GUI.

A fast and efficient port scanner developed in Python. PyScan is designed to quickly identify open ports and services on a target system, a crucial first step in any network security assessment.
